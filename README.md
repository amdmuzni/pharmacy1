# Pharmacy Management Website

## Overview of the Prject
This project is a simple web-based pharmacy system that allows
users to view available medicines, select quantities, enter
delivery details and place orders online.

## Features
- View available medicines with prices and stock quantity
- Select medicine type and quantity during checkout
- Enter delivery details
- Choose a payment method
- Order confirmation popup with delivery date and time

## User Flow
1. User views available medicines on the main page
2. User selects medicines and quantities
3. User enters delivery information
4. User selects a payment option
5. System displays order confirmation with delivery schedule

## Technologies Used
- HTML
- CSS
- JavaScript

## Scope of the Project
- This is a front end demonstration project
- No real payment gateway is integrated
- No real customer data is stored
